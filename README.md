# 🚦 Road Accident Dashboard

##  Overview
This project is an interactive Power BI dashboard created to analyze road accident trends from 2023–2026. The dashboard provides insights into accidents, fatalities, vehicle types, weather conditions, road types, and major accident causes.

---

##  Tools Used
- Power BI
- Excel

---

##  Features
- Year-wise accident trend analysis
- Vehicle type analysis
- Weather condition analysis
- Road type analysis
- Cause-wise accident analysis
- Interactive dashboard visualizations

---

##  Key Insights
- Overspeeding caused a high number of accidents
- Fog and rainy weather increased accident risks
